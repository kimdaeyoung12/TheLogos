분포 위를 걷는 산책자

핵심: MCMC는 직접 계산하기 어려운 분포에서 샘플을 얻기 위해, 높은 확률 영역에 더 오래 머무르는 마코프 체인을 만든다.

제안: x' = x + noise 수락확률: α = min(1, target(x') / target(x)) 오래 걷기: 방문 빈도 ≈ 목표 분포

제안 폭이 너무 작으면 천천히 움직이고, 너무 크면 대부분 거절된다.
