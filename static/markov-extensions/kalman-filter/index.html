예측과 측정의 균형

핵심: 칼만 필터는 예측값과 측정값을 단순 평균하지 않는다. 각 값의 불확실성을 보고 더 믿을 만한 쪽에 더 큰 가중치를 준다.

예측: x̂⁻ = F x̂, P⁻ = F P Fᵀ + Q 갱신: K = P⁻Hᵀ(HP⁻Hᵀ + R)⁻¹ x̂ = x̂⁻ + K(z − Hx̂⁻)

R이 커지면 센서를 덜 믿고, Q가 커지면 모델 예측을 덜 믿는다.
